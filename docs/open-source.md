# Open Source Contributions

I really enjoy tinkering with code and contributing to open source projects.
Here are some of the repositories I have created and contributed to.

Created by me:  

- [latent-calendar](https://williambdean.github.io/latent-calendar/): Analyze and model weekly calendar distributions using latent components
- [conjugate-models](https://williambdean.github.io/conjugate/): A Python package for Bayesian Conjugate Models
- [frame-search](https://williambdean.github.io/frame-search/): A Python package for searching DataFrames with GitHub search-like syntax
- [pandas-bootstrap](https://williambdean.github.io/pandas-bootstrap/): A Python package for bootstrapping in Pandas
- [lyft-bikes](https://williambdean.github.io/lyft-bikes/): A Python client of Lyft's Bike Share Data
- [closing-labels](https://github.com/williambdean/closing-labels): GitHub Action to copy labels from any Issues closed by a pull request into the pull request itself
- [but-better](https://williambdean.github.io/but-better/): Your Code, But Better - Python decorator for long running functions

Core contributor to: 

- [PyMC-Marketing](https://github.com/pymc-labs/pymc-marketing): Bayesian marketing toolbox in PyMC
- [octo.nvim](https://github.com/pwntester/octo.nvim): A Neovim plugin for interacting with GitHub Issues, Pull Requests, and other objects
